Introduction to Git - GitHub

Room 110 - 4:30 PM

Git is a cornerstone of the current programming paradigm. It is a tool which allows programmers to work together effectively and back up their code in a meaningful manner. What GitHub, the company hosting this workshop, did to the world of "version control" (i.e. Git) is very similar to what Google Docs did to traditional Word documents.

In this workshop, you'll be learning how to use Git and GitHub, some of the most important tools used by modern programmers. Not only are these skills extremely useful in the context of a single hackathon, but they are vital in being a successful programmer moving forward.
